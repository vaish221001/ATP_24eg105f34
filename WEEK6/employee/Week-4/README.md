# Week 4 - Basic HTML and CSS

This repository contains exercises from Week 4, focusing on the fundamentals of HTML structure, text formatting, lists, links, images, and basic CSS styling. 

## Project Structure

The `Week-4` directory includes the following files:

### 1. `Assignment1(first program).html`
A basic introductory HTML page demonstrating the fundamental structure of a web document.
- **Concepts Covered:**
  - Standard HTML5 boilerplate (`<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`).
  - Text formatting tags (`<b>` for bold, `<i>` for italics).
  - Headings (`<h1>`) and paragraphs (`<p>`).
  - Hyperlinks using the anchor tag (`<a>`).
  - Unordered lists (`<ul>`) and list items (`<li>`).
  - Nested lists (lists within lists).

### 2. `Assignment2(pochi).html`
A slightly more complex webpage that serves as a profile for "Pochi the Cat", incorporating an external stylesheet and media.
- **Concepts Covered:**
  - Linking to external CSS files using the `<link>` tag.
  - Using headings (`<h1>`, `<h2>`) for content organization.
  - Embedding images using the `<img>` tag.
  - Creating ordered (`<ol>`) and unordered (`<ul>`) lists.
  - Using semantic text formatting (`<i>`).
  - Using horizontal rules (`<hr>`) as thematic breaks.
  - Using `class` attributes (e.g., `class="footer"`) for targeted styling.

### 3. `style.css`
The external stylesheet used to style the "Pochi the Cat" profile page (`Assignment2(pochi).html`).
- **Concepts Covered:**
  - Selectors: Targeting elements like `body`, `h1`, `h2`, `p`, `img`, `ol`, `ul`, `a`, `hr`, and classes (`.footer`).
  - Typography: Setting `font-family`, `font-size`, `font-weight`, and `line-height`.
  - Box Model & Layout: Using `margin`, `padding`, `border`, and `float`.
  - Colors: Setting `color` and `background-color`.
  - Links: Styling links with `text-decoration` and colors.

## Getting Started

To view the assignments, simply open the HTML files in any modern web browser:

1. Clone the repository.
2. Navigate to the `Week-4` directory.
3. Double-click on `Assignment1(first program).html` or `Assignment2(pochi).html` to open them directly in your default browser.
