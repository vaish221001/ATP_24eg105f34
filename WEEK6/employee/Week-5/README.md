# Week 5: Advanced HTML, CSS & Introduction to React

This repository contains the assignments and projects completed during **Week 5** of the training program. The week's curriculum focused on building complex layouts with HTML and CSS, and introduced the fundamentals of React and Tailwind CSS.

## Directory Structure

The `Week-5` directory is organized into the following daily folders:

- **`Day-12/`**: Advanced HTML and basic CSS layouts.
- **`Day-13/`**: Responsive design and grid layouts.
- **`Day-14/`**: Introduction to React and Tailwind CSS.

---

## Detailed Breakdown

### Day 12

*   **`Assignment-1/`**: Focuses on advanced HTML table structures. Demonstrates data visualization using nested tables, rows, `colspan`, and `rowspan` attributes to create complex layouts like a grade book and a user directory.
*   **`Assignment-2/`**: Features a basic HTML and CSS layout implementing a blog card design. The cards include thematic tags (Travel, Technology, Fashion), titles, descriptions, and images, demonstrating fundamental CSS styling and positioning.
*   **`Assets/`**: Contains various images used within the Day 12 assignments.

### Day 13

*   **`Assignment-1/`**: A "Solo Leveling" themed webpage. It includes a hero section and a grid-based card layout, reinforcing concepts of semantic HTML and responsive CSS design.
*   **`Assignment-2/`**: A landing page layout featuring a hero section with text and an image, alongside a three-column grid displaying feature cards. 
*   **`Assignment-3/`**: A "Best City Guide" landing page. It showcases a clean layout with a navigation bar, a hero section, and structured content columns describing city features, culinary delights, and transportation.

### Day 14

*   **`Tailwind/`**: Contains the initial setup and configuration files for a Tailwind CSS project, demonstrating how to integrate the utility-first CSS framework into a modern development environment.
*   **`react-app-1/`**: A React application bootstrapped with Vite. It displays a dynamic product grid by rendering a reusable `<Product />` component, illustrating the concept of passing `props` and rendering lists in React.
*   **`react-app-2/`**: Another React Vite application that implements a structured layout using components. It includes a `<Navbar />`, a `<Footer />`, and a `<UsersList />` component that takes user data as props to dynamically render a list of users.

---

## Technologies Used

*   **HTML5:** Semantic markup, complex tables, forms, and structuring.
*   **CSS3:** Flexbox, Grid, responsive layouts, styling, and animations.
*   **React:** Component-based architecture, JSX, props, and rendering lists.
*   **Vite:** Frontend tooling and development server.
*   **Tailwind CSS:** Utility-first styling framework.

## How to Run the React Applications

To run either `react-app-1` or `react-app-2`:

1.  Navigate to the respective directory (e.g., `cd Day-14/react-app-1`).
2.  Install dependencies using `npm install`.
3.  Start the development server with `npm run dev`.
4.  Open the provided localhost URL in your browser to view the application.
