# Week-6 Learning Journey

Welcome to the **Week-6** repository! This week is dedicated to learning and building full-stack applications and exploring modern frontend and backend development concepts. The focus is on the MERN stack (MongoDB, Express.js, React.js, Node.js) along with various tools like Vite, Tailwind CSS, and state management libraries.

## 📂 Repository Structure

This repository is organized into several distinct projects to isolate different learning objectives and applications:

### 1. `React app-4`
A standard React application setup using Vite, Tailwind CSS, and React Router.
- **Tech Stack:** React, Vite, Tailwind CSS, React Router.
- **Purpose:** Exploring modern, fast React project initialization and routing with utility-first styling.

### 2. `editCounter`
A targeted React practice application.
- **Sub-folder:** `reacttask`
- **Tech Stack:** React, Vite, Tailwind CSS.
- **Purpose:** Hands-on practice with React state management and building interactive UI components like a counter.

### 3. `employee`
A full-stack Employee Management Application without JWT.
- **Frontend:** React, Vite, Tailwind CSS, Zustand (for state management), React Hook Form, Axios.
- **Backend:** Node.js, Express.js, MongoDB (Mongoose), bcryptjs (for password hashing).
- **Purpose:** Understanding basic full-stack integration, building RESTful APIs, connecting a React frontend to an Express backend, and using Zustand for global state management.

### 4. `employees`
An advanced full-stack Employee Management Application with Authentication.
- **Frontend:** React (`frontend-4`), Vite, Tailwind CSS, React Hook Form, React Router.
- **Backend:** Node.js (`Backend-4`), Express.js, MongoDB (Mongoose), JSON Web Tokens (JWT), bcrypt, cookie-parser.
- **Purpose:** Implementing robust user authentication using JWT and cookies. Demonstrates secure API endpoints, protected routes on the frontend, and full CRUD operations.

## 🚀 Key Technologies Used

- **Frontend:** React 19, Vite, Tailwind CSS, Zustand, React Hook Form, React Router v7.
- **Backend:** Node.js, Express.js, JWT, bcrypt.
- **Database:** MongoDB (via Mongoose).

## 🛠️ Getting Started

To run any of the projects locally, navigate to the specific project's frontend or backend directory and follow these standard steps:

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start Development Server:**
   - For React (Frontend) projects:
     ```bash
     npm run dev
     ```
   - For Node.js (Backend) projects:
     ```bash
     node server.js # or whatever script is defined
     ```
   
*(Make sure to set up appropriate `.env` variables for the backend projects involving MongoDB and JWT.)*
